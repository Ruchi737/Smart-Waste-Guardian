Smart Waste Guardian is a smart waste management web solution designed to improve waste segregation awareness and support cleaner cities. Users can select the waste type and enter their address, and the system instantly provides correct disposal guidance along with a pickup registration concept. It promotes proper handling of Dry Waste, Wet Waste, Plastic Waste, E-Waste, and Medical Waste.

This solution demonstrates how AI + simple technology can support sustainability, reduce unmanaged garbage, and help civic bodies plan better waste management in future scalable versions.
